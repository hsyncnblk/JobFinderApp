# JopFinderApp


## Installation

This project is a mobile application developed using React Native CLI. 

```bash
git clone https://github.com/hsyncnblk/JobFinderApp.git
```

```bash
npm install
```

```bash
npx react-native start
```

If you have problems during Gradle configuration:

```bash
cd android
```

```bash
./gradlew clean
```

```bash
./gradlew build
```
